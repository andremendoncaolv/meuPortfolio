# Personal Portfolio — André Mendonça

A personal portfolio website built with HTML, CSS, and JavaScript, showcasing my professional background, technical skills, and contact information.

🔗 **Live demo:** [portifolioandremendonca.netlify.app](https://portifolioandremendonca.netlify.app/)

## About

Static portfolio site designed to present my profile as a software developer to potential clients and employers. Clean layout with sections for bio, skills, and contact links.
